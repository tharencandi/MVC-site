Control Flow and Loops

Control Flow

In javascript, we have access to the same control flow found in most other languages, for example the if,else if and else blocks, as well as switch statements. 

if (condition_1) {

  statement_1; 

} else if (condition_2) {

  statement_2;

} else if (condition_n) {

  statement_n;

} else {

  statement_last;

}

Special Values Needed in Control FLow

These values all evaluate to false, while everything else evaluates to true.

  • false
  • undefined
  • null
  • 0
  • NaN
  • the empty string ("")

Loops

Just as in other languages, Javscript has for loops and while loops. The follow are all types of for loops in Javascript

  • for - loops through a block n times
  • for/in - loops through each of an objects properties
  • for/of - Loops through an iterable
