Variable Declerations

Three options to declare a variable:

var x = 1
let y = 1
const z = 1

You should be using let and const, as var is no longer needed in almost all cases (It is has been replaced by the more modern let and const).
The "let" keyword declares a block-scoped, local variable, optionally initializing it to a value. 

let a 
let x; 
x = "hello"

The const keyword declares a block-scoped, read-only named constant. Note that const is like python properties of reference types i.e objects are not protected, only the name (reference) 

const h = "constant"

What is Variable Scope

When you declare a variable outside of any function, it is called a global variable, because it is available to any other code in the current document. When you declare a variable within a function, it is called a local variable, because it is available only within that function.
