Basic Syntax

Objects

Nearly all objects in javascript are instances of "Object", which has some basic methods and properties.

Some other Data Types include:

  • Boolean: true and false.
  • null: A special keyword denoting a null value.
  • undefined: A top-level property whose value is not defined.
  • Number: An integer or floating point number
  • BigInt: An integer with arbitrary precision. For example: 9007199254740992n.
  • String: A sequence of characters e.g. "Hello"

Creating Objects

We create a javascript object much like a variable except we use javscript object notation.

const obj = {
name: "lauren",
height: 180.2,
unit: "cm"
}
