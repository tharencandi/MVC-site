CSS Selectors

Selectors in CSS are used to target HTML elements on our web pages that we want to style.
A CSS selector is the first part of a CSS rule. It tells the browser which HTML elements should be selected to have the CSS rule applied to them. The `h1` and `p` in the code block below are both selectors.
Put this into your stylesheet file and see what changes.

h1 {
color: red;
background-color: black;
}

p {
color: orange;
}

In this article, you will learn about different selectors that target HTML elements in different way and also cascade, specificity and inheritance in CSS. However, this article will not describe all selectors available, refer to the selectors reference table at the end of the page for an overview of all the selectors.

Selectors

Type Selectors: Targets an HTML element, for example, `<h1>`.

h1 {
color: cyan;
font-size: 25pt;
}

Class Selector: Targets a class in the HTML document. The syntax for the selector is `.` followed by the class name. `.replace_with_class_name`
A snippet from the template HTML document.

<p class="a_class">A paragraph with class.</p>

.a_class { }

ID Selector: Targets an id in the HTML document. The syntax for the selector is `#` followed by the id. `#replace_with_id`
A snippet from the template HTML document.

<span id="unique">span element</span>

#unique { }

Attribute Selector: Targets elements based on the presence of certain attribute on an element. `a[attr] { }` Based on out template, we have 2 `<a>` elements but with different `href` attributes. We can select one of them by using:

a[href="https://example.com"] { }

Pseudo Class Selector: Targets element based on the state of an element. For example, The syntax for this selector is the element, followed by `:` and then the state of the element. For example:

a:hover { }

Descendant Combinator: Style something based on their location in the HTML document. For example, to select an `<em>` nested inside the `<li>` element instead of the `<em>` in `<p>`, enter both element, separated by space as the selector.

<p> Lorem <em>Ipsum</em></p>
<ul>
<li>Item <span>one</span></li>
<li>Item two</li>
<li>Item <em>three</em></li>
</ul>

li em {
color: rebeccapurple;
}

The text colour of "three" will be `rebeccapurple` while "Ipsum" remains `black`.
