Applying CSS

To see how CSS can be used to style your HTML document, you can copy and paste this document into a plain text editor and save it as `index.html`. You can use and experiment with this template throughout the CSS guide.

<!DOCTYPE html>
<html lang="en">
<head>
<meta charset="UTF-8">
<title>Getting started with CSS</title>
</head>


<body>

<h1>Level 1 heading</h1>

<p>This is the first paragraph. In the text is a <span id="unique">span element</span> and also a <a href="https://example.com">link</a>.</p>
<p>This is the second paragraph. It contains an <em>emphasized</em> element and another <a href="https://duckduckgo.com">link</a>.</p>
<p class="a_class"> A paragraph with class.</p>
<ul>
<li>Item <span>one</span></li>
<li>Item two</li>
<li>Item <em>three</em></li>
</ul>

</body>
</html>

Next, create a file named `style.css` in the same directory as `index.html`. To link and apply the stylesheet `style.css` to `index.html`, add the following code anywhere between the `<head>` `</head>` in `index.html`.

<link rel="stylesheet" href="style.css">

The `<link>` element tells the browser that we have an external resources that we want to load and `rel` tells the browser the relationship between this external resources and the HTML document, in this case, the resources is a `stylesheet`. `href` informs the browser the location of the external resources. Since our stylesheet file is in the same directory as our `index.html`, we just need to enter the file name `style.css` as the value. Open `index.html` using your browser to see how your browser renders `index.html` with an empty stylesheet before we proceed to make changes. You would notice that your browser will do some default styling, where the headings are large and bold, and out list has bullets. This is because browsers have internal stylesheet which is allied to all pages by default. We can override the browser's styling by defining our own CSS rules.

p {
color: orange;
}
