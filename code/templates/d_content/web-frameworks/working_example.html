Working Example

CV website

Using The CV HTML website found in Basic Routes, we can apply all that we have learnt to create a functioning website.

Get final project here.

Project directory:

  • files (other static files)
  • images
  • styles (contains css and fonts)
  • templates (contains hmtl and tpl files)
  • webframe.py

webframe.py:


from bottle import route, run, template, static_file

@route("/styles/")
def serve_style(file):
    return static_file(file, root = "./styles")


@route("/images/")
def serve_image(file):
    return static_file(file, root = "./images")

@route("/files/")
def serve_static_files(file):
    return static_file(file, root = "./files")

# web pages:
@route('/')
@route('/home')
def hello():
    return static_file("index.html", root = "./templates")

@route('/resume')
def resume():
    return template("./templates/resume")


run(host='localhost', port= 8080, debug = True)
