Dynamic Routing

Adding CSS with a dynamic route

Bottle allows us to generalise routes to match multiple URL's. A dynamic route will accept anything between <>. For example @route("/helo/<name>") will accept /hello/Jerry as a URL.

<name> can then be passed as an argument to the corresponding function. This allows us to create a general function and route to load all CSS style sheets.

All html pages used in this example can be downloaded: example_project


@route("/styles/")
    def serve_style(file):
        return static_file(file, root = "./styles")

As well as all images:


@route("/images/")
    def serve_image(file):
        return static_file(file, root = "./images")
