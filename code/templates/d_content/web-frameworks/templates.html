Templates

Mix html with python!

Using the .tpl file extension, we can mix html code with python to add extra functionality to our webpages.

This syntactical mix can be seen below:


    %if name == 'World':
    <h1>Hello {{name}}!</h1>
    <p>This is a test.</p>
%else:
    <h1>Hello {{name.title()}}!</h1>
    <p>How are you?</p>
%end

This will store data inputted by the user in a form element 'username' and a form element 'password'. We can then use this data within python.
