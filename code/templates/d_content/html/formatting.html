HTML Formatting

The HTML language consists of headings which can be added to the web page to help structure it, this includes 6 different types of headings which vary in size: <h1> - <h6>. Although, the heading sizes are default, there is a way to customise the size of the heading through the style attribute. style="font-size:60px;"

This language also includes paragraphs to help structure the web page with <p>. The <p> adds a newline character and some white space under the text. However, the display of the web page depends on the screen and window size that the user would be looking through. With HTML, you cannot change the format by adding extra spaces or extra lines in your HTML code. The web browser automatically removes extra spaces and lines when the page is displayed.

HTML can add a horizontal line to create a break or separate content with <hr>, it can also break a line starting a new paragraph with <br>.

The style attribute uses the CSS property, to define the format of the HTML. One of the CSS properties that can be used is 'background-colour' which determines the background colour of the element. <body style="background-color:powderblue;"> <h1 style="background-color:powderblue;">This is a heading</h1>

Through the CSS property:

  • the colour of the text can also be defined: style="color:blue;"
  • The font size can be defined: style="font-family:verdana;"
  • The text-alignment can be defined: style="text-align:center;"

There are many different ways to display text such as:

  • To make the text bold: <b>
  • To make the text italic: <i>
  • To make the text deleted: <del>
  • To highlight a text: <mark>
  • To create a block around a text: <div>
  • To create a block that does not start a new line: <span>

Other ways to format text include quotation and citation:

  • To cite a book, poem, etc: <cite>
  • To define contact information: <address>
  • To add short quotation marks: <q>
