HTML Tags and Elements

HTML Classes

The attribute class usually outlines the class name in a style sheet, but it can also be used with javascript to modify elements within that class. Elements can be classified as a class to have common characteristics, this in terms of HTML, could mean a common font and colour of text. The syntax for a class can be denoted as a (.) before the class name, then curly braces {}. such as .city{ } <h2 class="city">London</h2> HTML elements can be associated with multiple classes, by a space to separate the class names.

.city{ } .main{ } <h2 class="city main">London</h2>
As stated above, these classes can be used in Javascript to perform tasks, which can be accessed with getElementswithClassName().

HTML ID's'

HTML consists of an id attribute that creates a unique id for each element where there shouldn’t be more than one element of the same id in a document. This attribute is used to specify a specific style declaration on the style sheet to use on that element. This can also be used to access that element in Javascript to use. The syntax of an id is a hash followed by the name of the id such as (#myHeader).

#myHeader { } <h1 id="myHeader">My Header</h1>
As noticed above Ids and classes seem very similar in HTML… The main difference between an id and class is that classes can be used for several elements but an id can only be used by one element in that document.
