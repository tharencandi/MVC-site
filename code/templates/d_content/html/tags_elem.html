HTML Tags and Elements

HTML Elements

A HTML element is defined with a tagname, the content of the web browser and then the ending tagname. The web browser reads the contents of the HTML file and displays them according to the tagname. Elements can be nested, meaning elements can be used in other elements. <tagname> Web Content </tagname> All HTML files needs a declaration of the type of document at the top defining it as the HTML5 document.

<!DOCTYPE html>

HTML Tags

The root element of the HTML page is defined by: <html> The contents about the HTML page is defined in the section: <head> The element that defines the title of the HTML page is, this would go under the <head> section: <title>

The next section contains the headings, paragraphs, images and any other content of the web page, defined as: <body> The large heading section is defined as: <h1>, which goes from <h1> to <h6> to most significant heading to the least significant heading. The paragraph section is defined as: <p> HTML elements with no content are called empty elements. The <br> tag is an empty element that defines a line break.
