HTML Attributes

HTML Attributes

HTML elements can have attributes, which provide additional information about the elements, usually with the syntax name='value'. The <a> syntax defines a hyperlink that can be added to the web page through the syntax: <a>, where the href attribute defines the links destination. <a href="https://www.w3schools.com"> This is a link</a> HTML can also define images that can be displayed in the web page with the <img> tag and the attributes:

  • The Src attribute refers to the source file
  • The alt attribute is the alternative text for an image which would be which would be displayed if the image cannot.
  • The width and the height attributes refer to the measurement of the image on the web page.

<img src="w3schools.jpg" alt="W3Schools.com" width="104" height="142">

The style attribute is used to add styles to the element (usually a text) such fonts, colours, etc. <p style="color:red;">This is a red paragraph.</p>

When using HTML, there is a way to define the language of the Web page to help the engines and browsers. This is done through the lang attribute which goes in the <html> tag. <html lang="en"> ('en' for english)

The title attribute defines information about an element, which appears when the user is about to click the title. <p title="I'm a tooltip">This is a paragraph.</p>

Tips

  • Use double quotations marks rather than single ones: "
  • Always put the attribute values in quotation: name="value"
  • Use lowercase attributes: title=
