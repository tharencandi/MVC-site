Welcome! This website aims to give you the knowledge to design and build your own website from the ground up!

What we cover

  • How the Web Works

    Learn how the internet functions, how websites are connected and how a client interacts with a web server.

  • HTML

    Learn the fundamental scripting language used to present text, videos, forms and images on a web page.

  • CSS

    Learn how to make web pages visually pleasing through custom style sheets!

  • Javascript

    Learn the number one object oriented programming language to create interactive environments on web page.

  • Web Frameworks

    Learn Bottle. The easiest way to get a taste of back end web dev and other tools to help you create bigger and better websites!
